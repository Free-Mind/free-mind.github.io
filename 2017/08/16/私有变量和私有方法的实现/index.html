私有变量和私有方法的实现

Posted on 2017-08-16 | In javascript

1. 利用WeakMap实现私有变量

① 将this作为weakMap的key值，当实例化后的对象被不再使用的时候，weakMap中的key也会被垃圾回收

② 在class外部实例化一个weakMap对象，这样，在a.js这个文件（模块）内部，_counter这个变量是可以被共享的，但是，在这个文件以外，就是不可见的。通过这种方式，实现私有变量；

③ 实现getter，setter方法，提供对私有变量的修改

2. 实现私有方法

同样，也是通过在class 外部定义方法，在b.js 的内部，方法是可以被访问的，但是在模块的外部，确实不可见的；

同时，通过bind，改变privateSay方法内部的this指向，从而可以操作实例的所有属性。
